# 🎫演劇公演のチケット予約システム
舞台「スペースなスペース」の公演を想定したチケット予約アプリケーションです。
座席表からリアルタイムに空席を確認し、お好みの席を予約することができます。

## 🪐プロジェクトの目的
「チケット予約という日常的な動作に、没入感のある体験を加える」ことを目標に、フロントエンドの視覚効果とバックエンドの整合性を重視して開発しました。

## ✨主要機能 (顧客側)
- **公演一覧**: カード形式のUIで、最新の公演情報を確認。
- **インタラクティブな座席選択**:
  - 動的に生成される座席レイアウト。
  - データベースと連動した「予約済み」座席のグレーアウト表示。
- **予約管理（マイページ）**: 自分が予約したチケットを一覧で確認可能。
- **レスポンシブデザイン**: スマートフォンからも操作しやすいUI設計。

## 🛠技術スタック
- **Framework**: Laravel 11 (PHP 8.4)
- **Frontend**: Blade / Tailwind CSS / Alpine.js (Breeze)
- **Database**: MySQL

## 💪こだわったポイント
- **予約一覧**: 顧客がログインし、自分の予約を一覧として確認できる機能
- **座席重複の防止**: データベースの制約とバリデーションを組み合わせ、同じ席が重複して予約されないように実装。

## 🚀今後の改善点
- **管理者画面の追加**: 顧客側だけでなく、管理者画面で予約管理や公演管理などを行えるようにする。
